# Challenge_Foro-hub
API Rest del foro-hub de alura, que contiene las funcionalidades CRUD para topicos, cursos y usuarios.
<img width="1166" height="318" alt="image" src="https://github.com/user-attachments/assets/dccd1e69-d493-4a8b-8f2a-ee40f9cedbae" />

*Versión 1.0

Tecnologías utilizadas:
- Java 17
- Spring Boot v3.5.4
- MySQL v8.0.36
- Swagger
